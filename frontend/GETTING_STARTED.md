# Frontend

## Olivia Smith Portfolio - Frontend

A professional portfolio website built with Vue.js 3.

### Quick Start

```bash
cd frontend
npm install
npm run dev
```

Then open `http://localhost:3000` in your browser.

### Features

- ✨ Modern, responsive design
- 🎨 Beautiful color scheme
- 📱 Mobile-first approach
- ⚡ Fast with Vite
- 🎯 Component-based architecture
- 🔗 Connected to Laravel API

### Sections

1. **Header** - Navigation bar
2. **Hero** - Profile introduction
3. **Services** - Design services offered
4. **Projects** - Portfolio projects
5. **About** - Professional info
6. **Journey** - Education & experience
7. **Footer** - Contact links

### Customization

Edit component files in `src/components/` to update content.

### Build for Production

```bash
npm run build
```

Output will be in `dist/` folder.
